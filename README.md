# nestjs-websocket-angular-example

## Description

example repo showing angular connecting to websockets on a Nestja web server

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ nodemon

# production mode
yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
